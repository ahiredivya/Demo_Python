# Demo_Python

This project demonstrates a simple backend-style application using Python and MySQL.
It includes CRUD operations, simulated API integration, and basic data visualization.

## Tech 
- Python
- MySQL
- Jupyter Notebook
- PyMySQL
- Pandas
- Matplotlib

## Features 
1. **CRUD Operations**  
   - **Create:** Insert new feedback records  
   - **Read:** Fetch all feedback records  
   - **Update:** Modify existing feedback ratings  
   - **Delete:** Remove specific feedback entries  

2. **Simulated API Integration**  
   - Demonstrates how to access and use data from a third-party API
     
3. **Reporting & Analytics**  
   - Calculates total feedback and average rating  
   - Prepares data for visualization

4. **Data Visualization**  
   - Bar chart showing distribution of feedback ratings
